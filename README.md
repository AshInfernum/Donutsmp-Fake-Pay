# 🍩 DonutSMP Fake Pay

<p align="center">
  <img src="https://i.imgur.com/9hw7rUV.jpeg" width="700">
</p>

A simple Minecraft mod that lets you **simulate /pay commands** and send **fake payments** to players.

Useful for trolling, testing server reactions, or creating content.

**I will share the modrinth page when the mod get approved.**

---

## ✨ Features

* Toggle **Fake Pay** with a keybind
* Send **any fake amount of money** to any player
* Simple and lightweight
* Works directly in-game
* Designed for quick use

---

## 📦 Requirements

You need the following mod installed:

* **Mod Menu**

Without Mod Menu the mod will still load, but **keybind configuration will be harder**.

---

## ⚙️ How To Use

1. Install the mod in your `mods` folder.
2. Launch the game.
3. Go to **Options → Controls → Keybinds**.
4. Set a key for **Fake Pay Toggle**.
5. Enable it and start sending fake payments.

---

## 📜 Example

```
/pay PlayerName 10B
```

The payment message will appear **as if the player received the money**, but no real transaction happens.

---

## ⚠️ Disclaimer

This mod only **simulates payment messages** and does not modify any real economy data.
